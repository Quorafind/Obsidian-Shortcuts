## Shortcuts

You can set up shortcuts for all commands.

Basically, it supports:

- With modifier: `Ctrl + V`
- Combo: `C + V`
- Single: `C`

And it shows: `a then c+b then d` when you press `a`, `c+b`, `d` in sequence.
